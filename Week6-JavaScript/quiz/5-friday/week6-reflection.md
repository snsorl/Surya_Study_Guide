# Reflection Quiz: JavaScript & TypeScript

This reflection quiz consists of 5 open-ended questions. Reflect on the material covered this week and write a short summary (2-3 sentences) for each question.

---

### Question 1: Surprising JavaScript Behaviors
What was the most surprising or counter-intuitive JavaScript behavior you discovered this week (e.g. type coercion, the event loop, or `typeof null`), and how will you prevent it from causing bugs in your code?

---

### Question 2: Transitioning from Java to ES6+ Classes
Compare your experience writing class inheritance and object methods in ES6+ with writing them in Java. What syntax differences did you find, and how do they map to JavaScript's prototype-based inheritance under the hood?

---

### Question 3: The Event Loop & Async Page Rendering
Explain how the browser's single-threaded Event Loop handles network fetch calls asynchronously. Why is it important to prevent page reloads during form submissions?

---

### Question 4: Migrating JavaScript to TypeScript
Reflect on the process of converting JavaScript files to TypeScript. How did declaring interfaces and annotating parameters help catch bugs, and what are the benefits of type-safety in collaborative teams?

---

### Question 5: Preparing for Project 3
Based on your experiences building the Project 2 frontend integration this week, how will you organize your project structure, package managers (npm), and compile configurations (tsconfig) as you prepare for Project 3?
